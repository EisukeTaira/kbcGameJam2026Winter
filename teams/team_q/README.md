# チーム紹介ページ作成マニュアル

KBC Game Jam 2026 Winterへの参加お疲れ様でした！
このテンプレートを使用して、あなたのチームのゲーム紹介ページを作成しましょう。

## 手順

### 1. 内容の編集 (`index.html`)
`index.html` をテキストエディタ（VS Codeなど）で開き、以下の箇所を変更してください。
`{{...}}` で囲まれている部分や、コメントで指示がある部分が対象です。

*   **タイトルとメタデータ**: `<title>` タグや `meta name="description"` を変更。
*   **ヘッダー**: チーム名や作品名に変更。
*   **ヒーローセクション**: メインのキャッチコピー、チーム名。
*   **Aboutセクション**: ゲームの概要、詳細、アピールポイント3つ。
*   **Galleryセクション**: スクリーンショット画像の差し替え。
*   **Downloadセクション**: Google Driveの共有リンクを設定。

### 2. 画像の差し替え
*   デフォルトでは `placeholder-img` というクラスでグレーのボックスが表示されています。
*   自分のゲームの画像を使用する場合は、`<img>` タグを使用してください。
    *   画像をフォルダ内（例: `images` フォルダを作成）に保存。
    *   HTMLを書き換え:
        ```html
        <!-- 変更前 -->
        <div class="placeholder-img">{{MainVisual}}</div>

        <!-- 変更後 -->
        <img src="images/my_game_visual.png" alt="メインビジュアル" style="width:100%; border-radius:8px;">
        ```

### 3. ダウンロードリンクの設定
Google Driveにゲーム（ZIPファイル）をアップロードし、「リンクを知っている全員」がダウンロードできるように共有設定を変更してください。
そのリンクを `index.html` の以下の部分に貼り付けてください。

```html
<a href="【ここにGoogle DriveのURLを貼り付け】" target="_blank" ...>
```

## デザインの調整
もしCSSが書けるメンバーがいれば、`style.css` を自由に編集して、ゲームの世界観に合わせてカスタマイズしてください！
色の変更（`--primary-color` など）だけでも雰囲気が変わります。

Good Luck!
